# Dental Clinic Management System

## 📖 Description
A comprehensive full-stack application designed to manage dental clinic operations. The system handles doctor schedules, patient records, and appointment bookings using modern web technologies and AI agents.

## 🛠️ Tech Stack
| Component | Technology |
|-----------|------------|
| **Frontend** | React.js, Tailwind CSS |
| **Backend** | Node.js, Express |
| **Database** | MongoDB, PostgreSQL |
| **DevOps** | Docker, Replit |
| **AI** | OpenAI Agents |
