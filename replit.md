# Dental Clinic Management System

Comprehensive full-stack application for managing dental clinic operations.

## Architecture
- **Frontend**: React + Vite + Tailwind CSS (Port 5000)
- **Backend**: Node.js + Express (Port 8000)
- **Database**: PostgreSQL (Replit Managed)

## Setup
- Workflows are configured for both frontend and backend.
- Database schema is initialized in PostgreSQL.
- Frontend is configured to proxy `/api` requests to the backend.

## Deployment
- Configured for autoscale deployment.
